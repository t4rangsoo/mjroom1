Web 페이지 만드는 방법

1. 크롬 브라우저로 github.com 접속하기 - 로그인 된 상태로 나타나게 됨 2. 왼쪽 위에 jjang217/mjroom1 클릭하기 3. Upload files - .html 파일과 이미지 파일 - 홈페이지 초기화면은 index.html 4. Html 파일 만들기 - 정보가 담긴 html 파일을 여러 개 만든다 - index.html 파일을 만들어서 하이퍼텍스트로 연결시킨다. - 쉽게 만드는 방법은 한글, 파워포인트로 페이지를 만들고, 다른이름으로 저장할 때 형식을 html 로 저장하면 됨 - https://www.w3schools.com/ 다양한 소스를 복사해서 사용할 수 있음 - 메모장 사용가능 : 다름이름으로 저장할 때 .html로 저장하기(모든 파일 선택 후) 5. 사이트 확인하는 방법 - Settings 클릭하기 - 하단에 GitHub Pages 웹페이지 주소 클릭하기 https://jjang217.github.io/mjroom1/ (나의 웹페이지 주소로 언제 어디서나 접속할 수 있음) * Source 박스 안에 marster branch 가 선택되어 있어야 함(확인만 하면 됨)
